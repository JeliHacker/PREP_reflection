I think the hardest thing that I've learned so far and actually understand is CSS selectors. At first they were a little complicated but then I got the hang of them. The important thing to remember with selectors is that commas act as "select this x and this y" and a space acts as "select all y within this x. A period selects a class and a # selects an id. That'll get you to the point where you can select about 90% of all things you need to select.
